# Описание проекта

Учебный проект интернет-магазина с использованием django фреймворка и bootstrap.

# Основные требования

## Работа с Git и GitHub

- [x] Домашка сдана через pull request из ветки домашней работы в ветку `develop`.
- [ ] В коммиты не добавлены игнорируемые файлы.

## Базовые настройки проекта

- [x] В проекте есть файл с зависимостями.

## Настройка проекта для работы с БД

- [x] Настройки DATABASES в проекте заменены на подключение к PostgreSQL, указаны ENGINE, NAME, USER, PASSWORD, HOST, PORT.
- [x] В зависимости добавлена библиотека `psycopg2` или `psycopg2-binary`.
- [ ] Файл с базой SQLite удален из репозитория.

## Сокрытие чувствительных данных

- [x] Настройки работы с БД вынесены в переменные окружения в файл `.env`.
- [x] Собран шаблон файла `.env`.

## Создание моделей

- [ ] Создана модель продукта.
- [ ] Модель продукта содержит необходимые поля.
- [ ] Создана модель категории.
- [ ] Модель категории содержит необходимые поля.
- [ ] В моделях категории и продукта описан класс `Meta`, указаны параметры `verbose_name` и `verbose_name_plural`.
