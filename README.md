# Описание проекта
Учебный проект интернет-магазина с использованием django фреймворка и bootstrap.

# Основные требования
## Работа с Git и GitHub

- [x] Домашка сдана через pull request из ветки домашней работы в ветку develop.
- [x] В коммиты не добавлены игнорируемые файлы.
- [x] Файл .gitignore создан в корне проекта, в файле учтены.idea, venv, __pycache__, env, базы данных sqlite.
- [x] В репозитории есть README-файл с описанием проекта, оформленный по правилам.

## Базовые настройки проекта

- [x] В проекте есть файл с зависимостями.Создан Django-проект.
- [x] Создано приложение catalog.
- [x] Приложение catalog зарегистрировано в INSTALLED_APPS.
- [x] Создан файл urls.py в приложении catalog.

## Разработка функциональности проекта

- [x] Создан HTML-шаблон главной страницы.
- [x] Элементы в шаблонах расположены упорядоченно.
- [x] Стили в шаблонах отображаются корректно.
- [x] Контроллер рендерит шаблон главной страницы функцией render.
- [x] Контроллер рендерит шаблон страницы контактов функцией render.
- [x] В контроллерах корректно указаны пути к шаблонам.
- [x] Контроллер главной страницы зарегистрирован на адрес / или home/.
- [x] Контроллер страницы контактов зарегистрирован на адрес contacts/.
- [x] В основном URL-файле использован include для подключения URL-файла приложения catalog.
- [x] Все URL-адреса в проекте заканчиваются на /.
